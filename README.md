# 🚲 Bike Sharing Data Analysis Dashboard

## 📌 Deskripsi Project

Dashboard ini dibuat untuk menganalisis pola penyewaan sepeda berdasarkan waktu (tren bulanan) dan tipe hari (working day vs holiday) menggunakan dataset Bike Sharing.

Analisis ini bertujuan untuk memahami pola penggunaan sepeda serta memberikan insight bisnis terkait peningkatan layanan transportasi berbasis sepeda.

---

## 📊 Fitur Dashboard

- Analisis tren penyewaan sepeda bulanan
- Perbandingan penyewaan pada hari kerja vs hari libur
- Distribusi jumlah penyewaan harian
- KPI utama (total penyewaan, rata-rata, maksimum)
- Insight otomatis berbasis data

## 🚀 Cara Menjalankan Dashboard 

### Setup Environment - Shell/Terminal
mkdir proyek_analisis_data
cd proyek_analisis_data
pip install -r requirements.txt

### Run Steamlit App
streamlit run dashboard/dashboard.py
